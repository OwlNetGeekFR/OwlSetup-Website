QUESTIONS FRÉQUENTES

Avant de commencer

La démonstration modifie-t-elle mon ordinateur ?

Non. Toutes les actions du site sont simulées. Seule l’application téléchargée pour votre système peut lancer les opérations après votre confirmation.

Pourquoi une autorisation administrateur peut-elle être demandée ?

Certaines installations, mises à jour et zones protégées nécessitent les droits administrateur. OwlSetup ne les demande que lorsque l’opération en a besoin.

Mes documents personnels sont-ils supprimés ?

Non. Le nettoyage protège vos documents et le dossier Téléchargements. La Corbeille n’est vidée que si vous la sélectionnez explicitement.

Quelle différence entre l’installateur et la version portable ?

L’installateur ajoute OwlSetup au menu Démarrer et facilite son lancement quotidien. La version portable fonctionne directement depuis le dossier de votre choix sans installation.

Pourquoi Windows peut-il afficher un avertissement ?

La signature de code est en cours de mise en place. Téléchargez uniquement OwlSetup depuis la Release GitHub officielle et comparez, si nécessaire, son empreinte avec le fichier SHA256.txt publié.

Quand Linux et macOS seront-ils disponibles ?

Aucune date n’est annoncée pour le moment. Les deux éditions sont en attente pendant que la version Windows stable reste prioritaire.
